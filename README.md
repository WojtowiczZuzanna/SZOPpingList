

## SZOPpingList 
### Aplikacja webowa


### Jak działa?
- Dodanie artykułu do listy zakupów
- Kategorie artykułów
- Zmiana ilości produktów
- Przerzucenie artykułu do listy produktów znajdujących się w domu (magazyn)
- Usunięcie produktu z list

### Funkcjonalność
1. Dane przechowywane są w bazie Cloud Firestore
2. Dane zawierają kolekcję produktów, opisywanych poprzez nazwę, rodzaj oraz ilość.
3. Aplikacja wyświetla produkty dostępne w bazie 
4. Informacje wyświetlane na ekranie dopasowane są do urządzenia mobilnego
